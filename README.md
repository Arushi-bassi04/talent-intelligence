# Talent Intelligence System

A talent analysis platform developed during the **IIT Kanpur × Eightfold AI Hackathon**.
The system focuses on analyzing developer profiles and generating structured insights to support more informed evaluation and decision-making.

---

## Features

* Analysis of developer/talent profiles
* Structured insight generation from raw data
* Automated evaluation logic
* Efficient data processing workflow

---

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python
* APIs: GitHub API (for data extraction)

## Setup Instructions

Clone the repository:

git clone https://github.com/Arushi-bassi04/talent-intelligence.git
cd talent-intelligence

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py
